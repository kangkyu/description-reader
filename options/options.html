‚ùì Help & Support

How do I get a Gemini API key?

Visit Google AI Studio, sign in with your Google account, and create a new API key. The service has a generous free tier for personal use.

Is the extension free to use?

Yes! The extension itself is completely free. You only need a free Gemini API key from Google, which includes generous usage limits for personal use.

Why isn't the summary button appearing?

Make sure you're on a YouTube video page (not the homepage or search results). The button appears near the video description. Try refreshing the page if you don't see it.

What if my API key isn't working?

Double-check that you copied the entire API key correctly. Ensure your API key has access to the Gemini API and hasn't exceeded its usage limits.
