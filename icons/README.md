# TubeBoost Icons

This directory should contain the extension icons:

- `icon16.png` - 16x16 pixel icon for browser UI
- `icon32.png` - 32x32 pixel icon for extension management
- `icon48.png` - 48x48 pixel icon for extension management
- `icon128.png` - 128x128 pixel icon for Chrome Web Store

These should be high-quality PNG files with the TubeBoost logo/brand colors.

For development, you can use simple colored squares or download icons from:

- https://www.flaticon.com
- https://heroicons.com
- Design custom icons with the purple/blue gradient brand colors

## Quick Setup

You can create simple placeholder icons by:

1. Using any image editor to create colored squares
2. Using the YouTube play button icon as inspiration
3. Incorporating the purple/blue gradient from the extension theme
