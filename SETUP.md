# YouTube Description Summarizer - Quick Setup

Get AI summaries of YouTube descriptions automatically!

## üöÄ Installation (2 minutes)

### Step 1: Install Extension

1. Download this extension folder
2. Open Chrome ‚Üí `chrome://extensions/`
3. Enable "Developer mode" (top right)
4. Click "Load unpacked" ‚Üí Select this folder
5. Pin the extension to your toolbar

### Step 2: Get API Keys (Free)

#### Gemini AI Key

1. Visit [Google AI Studio](https://makersuite.google.com/app/apikey)
2. Sign in with Google account
3. Click "Create API Key"
4. Copy the key

#### YouTube Data API Key

1. Visit [Google Cloud Console](https://console.cloud.google.com/apis/credentials)
2. Create new project (or select existing)
3. Enable "YouTube Data API v3"
4. Create API Key credentials
5. Copy the key

### Step 3: Configure Extension

1. Right-click extension icon ‚Üí "Options"
2. Paste both API keys
3. Click "Test" for each key to verify
4. Save both keys

## ‚úÖ Done!

Now visit any YouTube video with a description and you'll see an AI summary appear automatically in the top-right corner!

## ‚ùì Troubleshooting

**Summary not appearing?**

- Make sure both API keys are configured
- Check that video has a description
- Very short descriptions won't be summarized

**API key errors?**

- Test each key in extension options
- Make sure YouTube Data API v3 is enabled in Google Cloud
- Check API key hasn't exceeded quota
